О герое

Zeus — маг, который карает своих врагов молниями с небес. Его сила — чистая разрушительная энергия. Настоящий бог грома, один из лучших нукеров в Dota 2.

Способности

Arc Lightning: Молния, прыгающая по врагам.

Lightning Bolt: Мгновенный удар молнии по одной цели.

Static Field: Каждая атака наносит урон в процентах от здоровья.

Thundergod’s Wrath: Уничтожает врагов по всей карте!
